# Proof of concept for react native web using typescript

## install

```sh
npm i -g yarn

git clone git@github.com:nialloc9/react-native-web-typescript.git

cd react-native-web-typescript

yarn install
```

### run

```sh
    yarn start:ios
```

```sh
    yarn start:android
```

```sh
    yarn start:web
```

### lint

```sh
    yarn lint
```

### test

```sh
    yarn test
```

### deploy

```sh
    yarn build
```
